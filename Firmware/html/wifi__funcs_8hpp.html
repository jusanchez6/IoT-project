Detailed Description

Funciones para la gestión de WiFi, AP cautivo, sincronización NTP y comunicación segura (MQTT/HTTPS).

Este archivo contiene las funciones necesarias para configurar un Access Point, solicitar credenciales, conectarse a WiFi, sincronizar la hora, comunicarse con un broker seguro y preparar mensajes JSON de telemetría.
