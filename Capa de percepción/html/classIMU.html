Detailed Description

Clase para manejo del sensor MPU6050 con estimación de velocidad.

Ofrece funciones para inicializar y configurar el sensor, adquirir datos de aceleración, aplicar filtros y estimar la velocidad mediante integración numérica y filtros Kalman y de media móvil.
