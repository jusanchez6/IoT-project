Detailed Description

Ejemplo de integración de GNSS, IMU, LED y Alarma con FreeRTOS en un ESP32-S3.

Este programa corre en un ESP32-S3 DevKit y gestiona diferentes módulos de percepción y actuación mediante tareas en FreeRTOS:

  • GNSS (GPS) para posición, altitud y velocidad.
  • IMU (MPU6050) para aceleración y estimación de velocidad.
  • LED RGB (NeoPixel) como indicador de estado.
  • Alarma (buzzer o pin digital) para notificaciones.

Cada módulo corre en su propia tarea, lo que permite concurrencia y control en tiempo real.
